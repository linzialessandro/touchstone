### Why this name

A **touchstone** is the classical stone used to assay gold — a criterion of genuineness, not a hype scoreboard. It fits:

- **Honest measurement** (executable tests, not vibes)
- **Assay of open models** against a fixed standard
- **Not slop** — real English, short, memorable, academic-adjacent without cosplay

### Tagline options

1. *Touchstone — open models, closed-book code, open tests.*  
2. *Does the code run? Touchstone grades the only answer that matters.*  
3. *A coding touchstone for open-weight models.*
